AI Prompt Benchmark

Objective

This repository compares responses from different AI models using identical prompts and an identical evaluation framework.

The purpose is to evaluate:

*Technical Accuracy

*Completeness

*Educational Value

*Organization

*Safety

*Prompt Adherence

Models are evaluated using the same prompt and the same scoring criteria.
No model receives special treatment.
